# Message in a Bottle Protocol

Fleet agents communicate asynchronously through directory-based bottles.

## Directory Structure
- `from-fleet/` — Messages from other fleet agents
- `for-fleet/` — Messages for any fleet agent to pick up
- `for-oracle1/` — Messages specifically for Oracle1

## Naming Convention
`YYYY-MM-DD_TOPIC.md`

## Status
Active participant in the SuperInstance fleet.
