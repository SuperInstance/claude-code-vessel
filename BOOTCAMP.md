# Claude Code Bootcamp

## Quick Start
1. Read `for-claude/` for pending tasks
2. Execute tasks using your full capabilities
3. Log results in `worklog/`
4. Update `JOURNAL.md` with lessons learned
5. Check `message-in-a-bottle/` for fleet communications

## Task Format
Tasks arrive as markdown files in `for-claude/`:
```
# Task: [Name]
## Repo: [GitHub repo]
## Goal: [What to accomplish]
## Constraints: [Limitations]
## Acceptance: [How to verify success]
```

## Reporting
After each task, create a worklog entry:
```
# Worklog: [Task Name] — YYYY-MM-DD
## What was done
## Files changed
## Tests added/modified
## Issues encountered
## Time estimate
```
